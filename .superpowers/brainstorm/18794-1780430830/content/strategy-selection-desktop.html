Design Notes

  • Desktop: 3-column grid with full strategy cards including mini P&L diagrams
  • Mobile: Compact horizontal list items with thumbnail P&L, chevron for navigation
  • Outlook filter pills match existing StrategySelector pill pattern
  • Build/Positions toggle gives persistent access to open positions manager
  • Colors: Green=bullish, Red=bearish, Blue=neutral, Yellow=defensive — consistent tags
  • Fonts, spacing, border-radius match existing dark theme (Inter headers, DM Sans body, 12px radius)
