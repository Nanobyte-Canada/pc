Mobile — unchanged from v1

Mobile layout remains the same (Calls/Puts toggle, simplified chain, floating bottom bar). See v1 for full mobile mockup.

v2 — Column rebalancing:

  • Split ratio changed: Chain 55% / Sidebar 45% (was ~70/30). Sidebar now has room to breathe.
  • Chain simplified: Removed Volume and Last columns — now 7 columns (Bid, Ask, Delta | Strike | Delta, Bid, Ask) instead of 11. Less cramped, larger tap targets on bid/ask.
  • Leg cards expanded: 3-column grid layout (Strike, Mid, Expiry) with proper labels and spacing instead of cramped inline text.
  • P&L metrics: Each metric in its own mini-card with background tint — larger font sizes (16px vs 13px), more whitespace.
  • P&L chart: Taller (70px vs 60px) with bigger breakeven markers.
  • Breakeven: Shown in a styled row with background instead of plain text.
