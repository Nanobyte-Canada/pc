Design Decisions

  • Connections page: Connected accounts as horizontal cards with broker icon, status dot, sync time, value, and actions. Error states show amber border + Reconnect button.
  • Available brokers: Card grid showing broker icon, name, account types, and connection status.
  • Account Detail: Breadcrumb nav, account header with broker icon + status, 4-column KPI row (Total Value highlighted, Investment/Cash with currency breakdown, Returns with ROI/IRR).
  • Tab interface: Positions / Activities / Dividends tabs below KPIs. Positions show the same AG Grid pattern as the dashboard.
  • Mobile Connections: Stacked cards, tap to navigate to detail. Error state shows inline Reconnect pill.
  • Mobile Account Detail: Back arrow + breadcrumb, 2x2 KPI grid (Total Value + Returns top row), compact position list.
