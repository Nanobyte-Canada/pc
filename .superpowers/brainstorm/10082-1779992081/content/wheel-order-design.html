Current vs Enhanced

Current (static)
  • Click position → static info dialog
  • Shows strike, expiry, P&L (read-only)
  • "Close Position" button → instant close
  • No price input, no quantity, no order type
  • No account selection
Enhanced (interactive)
  • Click position → order form with live quote
  • Editable: option type, expiry, strike dropdowns
  • Quantity stepper (−/+)
  • Order type (Market/Limit), limit price input
  • Account selector dropdown
  • Estimated order total
  • Buy / Sell action buttons

Order Flow Design Decisions

  • Desktop: Side panel — slides in from the right (340px) when a position is clicked. The wheel grid compresses to make room. Panel stays open until closed with X. Pre-fills with clicked position's contract details.
  • Mobile: Bottom sheet — slides up over the dimmed grid. Scrollable if content exceeds viewport. Drag handle to dismiss.
  • Live quote section: Symbol, price, change. Bid × Size / Mid / Ask × Size in 3 cards (same pattern as Questrade reference).
  • Editable contract fields: Option type (Call/Put dropdown), Expiration (dropdown), Strike (dropdown). Pre-filled from the clicked position but changeable.
  • Quantity: Stepper with −/+ buttons and numeric center (same pattern as reference).
  • Order type: Market / Limit dropdown. When Limit selected, shows limit price input with currency selector.
  • Duration: Day / GTC dropdown.
  • Account selector: Shows current account with broker icon + masked number. Dropdown to switch.
  • Estimated total: Calculated from quantity × limit price × 100 (contract multiplier).
  • Buy / Sell buttons: Green tinted Buy, red tinted Sell. Both always visible — user picks action.
  • Opening new positions: Clicking the "+" button on the grid opens the same panel but with empty/default fields instead of pre-filled.
