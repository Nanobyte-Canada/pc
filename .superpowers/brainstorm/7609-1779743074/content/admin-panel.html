Design Decisions

  • Desktop: 6-column summary stats row (Total, Enriched, Pending, Quota with progress bar, Exchanges, Last Run). Instrument type breakdown in 6-column grid below. Workflows + Recent Runs side by side. Expandable run rows with chevron.
  • Mobile: Summary condensed to 2x2 grid (merged Total + Enriched into one card). Workflows as stacked cards with Run buttons. Recent Runs as expandable list.
  • Auto-refresh indicator: Green dot + "Auto-refresh 10s" in header. Manual Refresh button always visible.
  • Run states: Green dot = completed, red dot = failed, amber spinning = in progress. Failed runs show "Failed" label in red.
  • Admin accessed via "More" tab on mobile — only visible to ADMIN role users.
