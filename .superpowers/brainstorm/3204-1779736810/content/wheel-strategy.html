Design Decisions

  • Desktop grid: Expiry rows x Ticker columns. Position cards show type badge (CSP blue / CC amber), strike, premium collected, unrealized P&L. Empty slots have dashed "+" button. DTE badges: red (0-5d), yellow (6-20d), green (21+d). Totals row at bottom.
  • Capital summary: Horizontal bar with dividers — Available Cash (with C$/US$ breakdown), CSP Deployed, CCs Written, Total Premium, Unrealized P&L. Dual currency on each.
  • Mobile rethink: Grid doesn't work on mobile — positions are grouped by expiry date instead. Each expiry is a collapsible section with position cards listed vertically. Cards show ticker + type badge + strike + OTM% + premium + P&L.
  • Account selection: Desktop uses tab pills. Mobile uses a dropdown (no horizontal scroll).
  • Position cards clickable: Tapping opens ClosePositionDialog. Empty slots navigate to Options page with ticker pre-filled.
