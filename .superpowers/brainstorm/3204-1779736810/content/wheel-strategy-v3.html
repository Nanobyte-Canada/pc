Design Decisions

  • Desktop + button on occupied cells: Small emerald circle with "+" in the top-right corner of every position tile. Clicking navigates to Options page with that ticker pre-filled. Allows adding another position for the same ticker/expiry.
  • Empty cells: Still show the dashed "+" button in the center (unchanged).
  • Mobile FAB (Floating Action Button): Emerald gradient FAB in bottom-right corner above the tab bar. Tapping opens the Options page to create a new position. This is the primary way to add positions on mobile.
  • Mobile position cards: Tapping an existing card opens ClosePositionDialog (same as before). The FAB is always visible for adding new ones.
